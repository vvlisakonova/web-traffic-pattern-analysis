# Анализ трафика веб-сайта: выявление паттернов пользовательской активности

Анализ данных о трафике веб-сайта с целью выявления паттернов пользовательской активности. Включает предобработку данных, анализ временных и поведенческих метрик, кластеризацию пользователей методом K-means и визуализацию результатов.

---

## Описание

Задача: на основе данных о сессиях пользователей веб-сайта электронной коммерции выявить паттерны активности — временные, поведенческие, по источникам трафика и устройствам.

В ходе работы:
- проведён первичный анализ и предобработка данных
- выполнен анализ конверсии, источников трафика и типов устройств
- исследованы временные паттерны (по часам, дням недели, месяцам)
- проанализированы поведенческие метрики и корреляции
- проведена кластеризация пользователей методом K-means с визуализацией через PCA

## Датасет

Датасет сгенерирован синтетически и содержит 105 847 записей о сессиях пользователей веб-сайта электронной коммерции за период июнь–ноябрь 2025 года.

Файл `web_traffic_data.csv` приложен к репозиторию.

Основные поля:

| Поле | Описание |
|------|----------|
| `session_id` | Уникальный идентификатор сессии |
| `timestamp` | Время начала сессии |
| `traffic_source` | Источник трафика (organic, direct, social, paid, referral) |
| `device_type` | Тип устройства (mobile, desktop, tablet) |
| `geo_location` | География пользователя |
| `pages_viewed` | Количество просмотренных страниц |
| `page_duration` | Время на сайте (секунды) |
| `conversion` | Целевой признак: совершена конверсия (1) или нет (0) |

## Результаты

Ключевые паттерны, выявленные в ходе анализа:

- **Временные:** основной пик активности — 19:00–21:00, конверсия в выходные (6,29%) выше, чем в будни (4,87%). Резкий рост трафика в осенний период (сентябрь–ноябрь).

- **По источникам:** платная реклама лидирует по конверсии (7,66%), социальные сети — по объёму трафика, но с минимальной конверсией (3,00%).

- **По устройствам:** мобильный трафик преобладает (58,3%), однако конверсия на десктопе вдвое выше (7,20% против 4,15%).

- **Поведенческие:** сильная корреляция между глубиной просмотра и временем на сайте (r = 0,888). Пользователи с конверсией проводят на сайте в 1,5 раза больше времени.

- **Кластеризация:** выявлены 4 типа пользователей — «Глубокие исследователи», «Утренние посетители», «Целенаправленные покупатели» и «Вечерние быстрые посетители».
